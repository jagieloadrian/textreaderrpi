# Plan 05-03: Route Package Unification + Test Package Unification

## Objective

Move all routes from `com.anjo.web.routes.*` into `com.anjo.routing.*` (D5-07).
Unify all test classes under `com.anjo.*` sub-packages matching production code (D5-08).

## Background

Phase 3/4 left routes split between `web.routes` and `routing` packages. Decision D5-07 consolidates all routes into `com.anjo.routing.*`. Tests are currently split between `com.anjo.*` and bare `routing/` in the test source root. Decision D5-08 unifies all test packages to mirror production exactly.
