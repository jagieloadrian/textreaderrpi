# Plan 05-04: Schedule Data Model + Exposed/H2 Database Setup

## Objective

Define the `Schedule` data model with all fields from D5-10.
Set up Jetbrains Exposed + H2 persistence (D5-13): `DatabaseFactory`, `SchedulesTable`, `ScheduleRepository`.
Add new Gradle dependencies (Exposed, H2, HikariCP, cron-utils).
Wire database initialization into the Ktor application.

## Background

Plans 05-05 through 05-07 all depend on the Schedule model and ScheduleRepository existing. This plan creates those foundations. H2 is the default database (embedded, file-mode for production, in-memory for tests). PostgreSQL is supported via config.

  <action>
    Create `src/main/kotlin/com/anjo/db/ScheduleRepository.kt`.
    Package: `package com.anjo.db`
    Imports: `org.jetbrains.exposed.sql.*`, `org.jetbrains.exposed.sql.SqlExpressionBuilder.eq`, `org.jetbrains.exposed.sql.transactions.experimental.newSuspendedTransaction`, model imports

    Define `class ScheduleRepository` with these suspend functions (ALL using `newSuspendedTransaction { }`):

    `suspend fun findAll(): List<Schedule>` — `SchedulesTable.selectAll().map { it.toSchedule() }`

    `suspend fun findById(id: String): Schedule?` — select where id eq given id, return null if not found

    `suspend fun findAllActive(): List<Schedule>` — select where status eq "ACTIVE" or "PAUSED"

    `suspend fun insert(schedule: Schedule): Schedule`:
    - Generate UUID: `java.util.UUID.randomUUID().toString()`
    - Set `createdAt` to `java.time.Instant.now().toString()`
    - Insert row using `SchedulesTable.insert { it[field] = value }` pattern
    - Return the inserted Schedule with the generated id

    `suspend fun update(id: String, schedule: Schedule): Schedule?`:
    - `SchedulesTable.update({ SchedulesTable.id eq id }) { ... }`
    - Return updated record or null if not found

    `suspend fun updateStatus(id: String, status: String): Unit`:
    - Update only the status column for given id

    `suspend fun delete(id: String): Boolean`:
    - `SchedulesTable.deleteWhere { SchedulesTable.id eq id }`
    - Return `deletedRows > 0`

    Private extension `fun ResultRow.toSchedule(): Schedule`:
    - Convert each column to corresponding Schedule field
    - Use `.toString()` for enum conversions: `Effect.valueOf(this[SchedulesTable.effect])`
  </action>
