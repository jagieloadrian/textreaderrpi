## Objective
Create HTML template infrastructure (BaseLayout) and deliver GET / (text submission form) and GET /status (display info) pages using Ktor HTML DSL. Style with Pico CSS v2 from CDN.
## Tasks
### Task 1: Create BaseLayout template + IndexPage
- Create BaseLayout.kt (Ktor Template<HTML>)
- Link Pico CSS v2 from CDN (https://cdn.jsdelivr.net/npm/@picocss/pico@2/css/pico.min.css)
- Create IndexPage.kt with text submission form (textarea, character counter, submit button)
- Add WebRoutes GET / endpoint
- Responsive viewport meta tag, dark theme
### Task 2: Create StatusPage + app.js character counter + form submit
- Create StatusPage.kt showing driver type + hardware status
- Add WebRoutes GET /status endpoint
- Create app.js with updateCounter() (display character count) and submitForm() (POST to /api/text via fetch)
- Keep app.js <100 lines
- Add tests for both pages
